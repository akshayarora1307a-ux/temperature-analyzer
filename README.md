# 🌡️ Temperature Analyzer (NumPy)

## 📌 Overview

A simple GUI-based Python application that analyzes temperature data using NumPy.

---

## 🚀 Features

* Add temperature values one by one
* Calculate average, max, min, and standard deviation
* Clean GUI interface
* Reset data

---

## 🛠 Tech Stack

* Python
* NumPy
* Tkinter
